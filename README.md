# LEDSmile

This project uses Capacitors, Resistors and Transistors to blink 4 LEDs in a loop. The buttons start the blinking!
Use the 2 pin Connection header to connect to power.
To use, simplfy click the first switch the turn on the power, and the other switch the start the blinking of t he 4 LEDs
I made this to learn basic circutry and KiCad!
